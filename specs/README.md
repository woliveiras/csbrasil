# Especificações

Este índice é a fonte de verdade para a numeração, a família e o estado das
especificações do CS Brasil. A numeração é global, crescente e tem quatro
dígitos.

## Blocos de numeração

| Intervalo | Família | Finalidade |
| --- | --- | --- |
| 0001–0099 | platform | Engine, toolchain, exportação e infraestrutura do cliente |
| 0100–0199 | gameplay | Regras, modos, armas e comportamento dos atores |
| 0200–0299 | content | Mapas, personagens, áudio e conteúdo visual |

| Spec | Título | Status | Depends on |

| [0001](0001-godot-web-port/spec.md) | Port do jogo para Godot Web | Approved | — |

## Estados

- `Draft`: em elaboração ou aguardando aprovação.
- `Approved`: aprovada para implementação.
- `Implemented`: entregue e verificada.
- `Superseded`: substituída por outra especificação.
